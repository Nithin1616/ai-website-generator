# AI Website Generator

An AI-powered web app that generates complete frontend websites (HTML, CSS, JavaScript) from simple user input.

## Live Demo
https://your-app-link.streamlit.app

## Features
- Generate full websites using AI
- Live preview inside the app
- Download complete project files (ZIP)
- Clean modular architecture (services + utils)
- Error handling & safe parsing

## 🛠 Tech Stack
- Python
- Streamlit
- LangChain
- Google Gemini API
